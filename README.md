# Portafolio Personal — Juan Carlos Camacho

Bienvenido al repositorio de mi portafolio digital. Este proyecto ha sido desarrollado para exhibir mi trayectoria académica y profesional, así como mis principales proyectos, habilidades técnicas y filosofía de trabajo en el desarrollo de software.

## 🚀 Sobre el Proyecto

Este portafolio es un sitio web dinámico y optimizado construido para centralizar mi identidad digital. La información (proyectos, habilidades, contactos, etc.) se gestiona de manera modular a través de archivos JSON locales, lo que permite una actualización fácil y rápida del contenido sin alterar la estructura base de los componentes.

### 🛠️ Tecnologías y Herramientas

El proyecto está construido utilizando tecnologías web modernas enfocadas en el rendimiento y la escalabilidad de los componentes:

- **[Astro](https://astro.build/):** Framework principal para la generación de la estructura web estática y de carga superrápida.
- **TypeScript / JavaScript:** Modularidad, utilidades y lógica para la interfaz.
- **CSS3:** Diseño responsivo, tematización y uso de variables CSS globales y media queries para una mejor adaptación.
- **HTML Semántico:** Estructuración del contenido base con buenas prácticas de accesibilidad.

## 📚 Secciones del Portafolio

El proyecto está esquematizado de la siguiente manera:

- **Inicio:** Resumen de mi perfil profesional y filosofía de trabajo.
- **Proyectos:** Catálogo de softwares desarrollados (académicos y personales), desde extensiones de VS Code hasta aplicaciones móviles y algoritmos.
- **Conocimientos:** Listado de lenguajes de programación, frameworks, bases de datos y herramientas de desarrollo.
- **Curriculum Vitae (CV):** Versión estructurada en web de mi perfil integral.
- **Contacto:** Enlaces directos a mis redes profesionales.

## 👨‍💻 Sobre el Autor

Soy estudiante de último año de Ingeniería en Sistemas en la Universidad Nacional de Costa Rica (UNA).
